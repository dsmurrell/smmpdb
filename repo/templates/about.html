About

SMMPDB (Small Molecule Measured Property Database) is a collection of publicly available measured property data.

The data can be accessed on a per submission basis or can be accessed as a whole database download.


Anyone can submit a collection of measured property data by uploading a SMILES file along with a meta information file, the format of which can be found here.

Prediction models have been carefully constructed and are also made available here.
